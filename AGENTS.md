# MCP Gemini Design
Gemini is your frontend developer. For all UI/design work, use Gemini Design MCP tools.

## When to use Gemini Design MCP
- New visual component or layout: use `snippet_frontend` or `create_frontend`.
- Redesign existing UI: use `modify_frontend`.
- Pure logic or tiny text changes: handle locally.

## Remote MCP server
- URL: `https://gemini-design-mcp-server-production.up.railway.app/mcp`
- Auth: `Authorization: Bearer $GEMINI_DESIGN_API_KEY`
